## 🎭 Animations

### 1. Apparition des Questions (Stagger Effect)

```
Temps:  0ms     100ms   200ms   300ms   400ms   500ms
        ↓       ↓       ↓       ↓       ↓       ↓
Card 1: [Fade in + Slide up]
Card 2:         [Fade in + Slide up]
Card 3:                 [Fade in + Slide up]
Card 4:                         [Fade in + Slide up]
Card 5:                                 [Fade in + Slide up]
Card 6:                                         [Fade in + Slide up]

État initial : opacity: 0, y: +20px
État final   : opacity: 1, y: 0px
Durée        : 300ms par carte
Type         : spring (rebond naturel)
```

### 2. Hover sur Carte

```
Normal → Hover
  ↓        ↓
Scale: 1.0 → 1.05
Y:     0px → -5px
Shadow: md → xl
Border: gray-200 → purple-300
Duration: 300ms
Easing: ease-in-out
```

### 3. Clic sur Question

```
1. Tap animation (scale: 0.98)
   ↓ 50ms
2. Question envoyée
   ↓
3. Disparition de SuggestedQuestions
   ↓ 200ms
4. Apparition du message utilisateur
   ↓ 300ms
5. Apparition du typing indicator
   ↓
6. Streaming de la réponse
```

### 4. Typing Indicator Animation

```
Dot 1:  ●     ●     ●     ●
        ↓↑    ↓↑    ↓↑    ↓↑
Dot 2:    ●     ●     ●     ●
          ↓↑    ↓↑    ↓↑    ↓↑
Dot 3:      ●     ●     ●     ●
            ↓↑    ↓↑    ↓↑    ↓↑

Delay: Dot 1 (0ms), Dot 2 (+150ms), Dot 3 (+300ms)
Movement: Y: 0 → -8px → 0
Duration: 600ms
Repeat: Infinite
```

### 5. Message Entrance

```
Initial state:
  opacity: 0
  y: +20px
  scale: 0.95

Animate to:
  opacity: 1
  y: 0px
  scale: 1.0

Type: spring
Stiffness: 500
Damping: 30
```
